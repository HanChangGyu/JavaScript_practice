# 🌱 JavaScript Practice for React

리액트(React) 학습을 시작하기 전, 탄탄한 기본기를 다지기 위한 **자바스크립트 문법 실습 저장소**입니다.

기초 문법부터 리액트에서 자주 사용되는 ES6+ 심화 문법까지 코드를 작성하며 학습한 내용을 기록합니다.

## 🎯 학습 목표
- 자바스크립트의 핵심 동작 원리 이해하기
- 리액트 사용을 위한 ES6+ 최신 문법 숙달하기 (Arrow Function, Destructuring, Spread 등)
- 비동기 처리(Promise, async/await) 완벽하게 이해하기
